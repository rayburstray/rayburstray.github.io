前端知识合集

什么是Web服务器？

在 Python 中，Web 服务器是用于处理 HTTP 请求并将响应返回给客户端的程序。Python 提供了多种 Web 服务器选项，可以根据需求选择适合的工具。

WSGI是什么，ASGI又是什么？

ASGI 和 WSGI 是 Python Web 开发中用于连接 Web 服务器和应用程序的两种接口规范。它们的主要作用是定义服务器和应用程序之间的通信方式，但它们在设计理念和功能上有显著区别。

WSGI 是 Python 的 Web 服务器和应用程序之间的标准接口。
它于 2003 年提出，旨在统一 Python Web 开发中的服务器和应用程序交互方式。

ASGI 是 WSGI 的异步版本，旨在支持现代 Web 应用的异步特性。
它于 2016 年提出，是 Python 异步 Web 开发的标准接口。

uvicorn是什么？

Uvicorn 是一个轻量级、高性能的 ASGI（Asynchronous Server Gateway Interface）服务器，专门用于运行 Python 的异步 Web 应用程序。它是构建和部署现代 Python Web 应用（尤其是基于异步框架的应用）的常用工具

这里有CSDN介绍它的链接

loguru是什么？

Loguru 是一个 Python 日志库，旨在简化日志记录的过程，提供更直观、更强大的日志功能。与 Python 标准库中的 logging 模块相比，Loguru 更加易用且功能丰富，适合快速开发和调试
