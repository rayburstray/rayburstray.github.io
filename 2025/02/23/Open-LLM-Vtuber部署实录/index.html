为了成功地把动作映射修改对，我们有必要恶补一些live2d的专业知识（呃啊啊我还不是很了解这方面的）

pixi-live2d-display-lipsyncpatch

一个增强和修补 PixiJS 与 Live2D 显示功能的库或插件。特别专注于实现更精确的唇形同步（Lip Sync）效果。

PixiJS

是一个高性能的 2D 渲染引擎，适合用于 WebGL 和 HTML5 Canvas 渲染。

WebGL

WebGL（Web Graphics Library）是一种基于浏览器的3D图形渲染技术，它让开发者能够直接利用硬件加速（GPU）在网页上渲染复杂的2D和3D图形，而不需要安装任何额外的插件或软件。WebGL是作为一种低级JavaScript API设计的，并与HTML5的canvas元素紧密结合，为网络应用提供了强大的图形处理能力。
